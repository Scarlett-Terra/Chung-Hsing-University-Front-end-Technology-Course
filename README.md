Vue 3 前端作品集

這是我的個人履歷與前端作品集網站，主要用來整理我的前端學習成果、Vue 3 專案、JavaScript 互動作品、RWD 響應式切版練習，以及視覺設計作品。

網站中包含個人介紹、核心技能、作品列表、作品詳細頁與履歷頁，希望讓人事與技術主管可以快速了解我的能力與作品方向。

## 線上 Demo

👉 https://scarlett-terra.github.io/vue3-portfolio-website/

## 專案特色

- 使用 Vue 3 製作單頁式作品集網站
- 整理多個前端作品，包含 Vue、JavaScript、HTML、CSS、RWD
- 作品卡片可依技能分類篩選
- 作品詳細頁展示專案功能、技術亮點與 Demo 連結
- Photoshop / Illustrator 作品以圖片相簿方式呈現
- 履歷頁整合個人經歷、技能、聯絡資訊與 PDF 下載
- 加入 Open Graph，分享網址時可顯示專屬預覽圖片

## 使用技術

- Vue 3
- Vue Router
- JavaScript
- HTML / CSS
- RWD 響應式設計

- GitHub Pages
